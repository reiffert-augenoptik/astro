# Reiffert Augenoptik Astro website

## How to develop

Start by installing all requirements:

- [Node.js v24](https://nodejs.org/en)

Then, clone the project and run it.

```bash
git clone git@github.com:reiffert-augenoptik/astro.git
cd astro
npm run dev
```
